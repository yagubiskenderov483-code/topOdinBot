# 🎁 Gift Deals Bot

Телеграм бот для безопасного проведения сделок.

## Установка

1. Клонируй репозиторий:
git clone https://github.com/твой_ник/название_репо.git
cd название_репо

2. Установи зависимости:
pip install -r requirements.txt

3. Создай файл .env и заполни его (см. ниже)

4. Запусти бота:
python bot.py

## Настройка .env

Создай файл .env в папке с ботом и вставь:
BOT_TOKEN=твой_токен
ADMIN_ID=твой_телеграм_айди

## Функции

- Создание сделок (НФТ, Звёзды, Крипта, Premium и др.)
- Профиль пользователя
- Пополнение баланса
- Топ продавцов
- Админ панель
- Мультиязычность (8 языков СНГ)
